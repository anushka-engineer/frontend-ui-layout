Raga Designers - Frontend Project
-->Project Overview
This is a responsive homepage built using HTML, CSS, and JavaScript. It includes sections like header, hero, about, services, portfolio, testimonials, contact form, and footer.

--->Design Decisions
     - Used Flexbox and Grid for responsive layout
     - Kept UI clean and modern with proper spacing
     - Added simple hover effects and transitions
     - Focused on mobile-first responsive design

--->Challenges Faced
     - Making navigation responsive with hamburger menu
     - Handling image path issues in GitHub deployment
     - Aligning sections properly across mobile and desktop views
     - Faced difficulty in making the navigation menu responsive for mobile view.
     - Had issues with image path in GitHub deployment, which was corrected by fixing folder structure.
     - Managed layout adjustments for different screen sizes.

--->Future Improvements
    - Add more animations and smooth transitions
    - Improve UI with advanced design components
    - Add backend integration for contact form
    - Make portfolio dynamic using JSON data
